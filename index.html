Carnet de voyage - Les nourritures

Humanité - Xavier JEANNENEY

  • Pour voyager dans mon carnet vous pouvez utiliser les flèches directionnelles
  • Dans les déplacements horizontaux vous visiterez les différents mets que j'ai pu déguster
  • Les deplacements verticaux vous permettront de mieux comprendre comment j'ai vécu cette dégustation
  • En montant, on remonte dans la chaine d'engendrement des plats, ma réflexion se base sur la question :

    • "Que faut-il pour produire cette chose ?"

  • En descendant, vous descendrez dans la chaîne d'engendrement des plats, ma réflexion se base sur la question :

    • "Qu'est-ce que cette chose engendre ?"

  • La réflexion n'a pas pour but d'être exhaustive, mais d'interroger mon ressenti et ma compréhension de ce qui me nourrit et donc me construit
